# SQS Visibility Timeout

**메시지를 가져간 후** 다른 소비자가 보지 못하게 하는 시간입니다.

## 동작

- 메시지를 **Receive** 하면 Visibility Timeout 동안 **다른 소비자에게 안 보임**.
- 처리 완료 전에 **Delete** 하지 않으면 타임아웃 후 다시 노출(재처리 가능).

## 설정

- 타임아웃을 **처리 시간보다 길게** 두어 중복 처리 줄이기. 너무 짧으면 중복, 길면 실패 시 복구 지연.

## 요약

- Visibility Timeout = 수신 후 다른 소비자에게 숨기는 시간. 삭제 전 만료 시 재노출.
