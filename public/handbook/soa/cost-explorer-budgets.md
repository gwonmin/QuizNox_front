# Cost Explorer / Budgets

**비용·사용량 분석**(Cost Explorer)과 **예산·알림**(Budgets) 설정입니다.

---

## 특징

- **Cost Explorer**: 비용·사용량 시각화. **서비스·리전·태그**별 그룹, 일/월 단위. **비용 할당 태그**를 활성화해야 태그별 비용 조회 가능.
- **Budgets**: 예산 금액·예상 비용(forecast) 임계값 설정. 초과·예상 초과 시 **이메일·SNS** 알림. **Savings Plans 사용률** 알림도 Budgets에서.
- **Tag Editor**: 리소스 검색·태그 일괄 수정. “No Tagkey” 비용이 있을 때 **Tag Editor**로 미태그 리소스 찾아 태그.
- **팀별 예산 알림**: 팀(서비스)별 **Budgets** 생성, 예상 비용 임계값 + 수신자 설정.
- **과금 알림**: **관리/페이어 계정**에서 **Billing alerts** 활성화 필요. 멤버 계정만으로는 알림 설정 제한될 수 있음.

---

## 시나리오

| 시나리오 | 접근 |
|----------|------|
| 태그별 비용이 Cost Explorer에 안 나옴 | **비용 할당 태그** 활성화(관리 계정). 태그 적용 후 **24시간~수일** 후 반영. |
| “No Tagkey” 비용이 많을 때 | **Tag Editor**로 태그 없는 리소스 검색 후 태그 부여. |
| 팀별로 예산 초과·예상 초과 알림 | **Budgets**를 팀(또는 서비스 필터)별로 생성. 예상 비용 임계값 + 이메일/SNS. |
| Savings Plans 사용률 90% 미만 시 알림 | **Budgets**에서 Savings Plans 예산·사용률 임계값 설정. |
| 데이터 전송 비용 75% 도달 시 알림 | **Budgets**로 데이터 전송 비용 예산 생성, 75% 알림. |
| 프로젝트별 비용 보기 | **비용 할당 태그** 활성화 후 프로젝트 태그 부여. Cost Explorer에서 그룹 바이 태그. |

| 구분 | Cost Explorer | Budgets |
|------|---------------|---------|
| 용도 | 비용·사용량 분석·시각화 | 예산 설정·초과/예상 초과 알림 |
