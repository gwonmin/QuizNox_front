# CloudTrail vs Config 차이

**API 호출 기록** vs **리소스 설정·변경 이력**을 담당합니다.

## CloudTrail

- **누가, 언제, 어떤 API**를 호출했는지 이벤트 로그
- 감사·규정 준수·보안 분석
- 리전·글로벌 서비스 트레일 지원

## Config

- **리소스 설정 스냅샷·변경 이력**
- “리소스가 어떻게 바뀌었는지”, 규칙 기반 평가(Compliance)
- 원하는 리소스 타입만 기록 가능

## 요약

| 구분 | CloudTrail | Config |
|------|------------|--------|
| 기록 대상 | API 호출 이벤트 | 리소스 설정·변경 |
| 질문 | “무슨 API가 호출됐나?” | “리소스가 어떻게 바뀌었나?” |
