# SSE-S3 vs SSE-KMS

S3 **서버 측 암호화(SSE)** 의 두 방식입니다.

## SSE-S3

- **S3가 키 관리**: AWS가 키를 관리하고 암호화·복호화
- 별도 설정 최소, 버킷 기본 암호화로 설정 가능

## SSE-KMS

- **KMS 키** 사용: 고객 마스터 키(CMK)로 데이터 암호화
- 감사(CloudTrail), 키 정책·권한 제어 가능
- SSE-S3보다 비용·제한 있음

## 요약

| 구분 | SSE-S3 | SSE-KMS |
|------|--------|---------|
| 키 관리 | S3 | KMS |
| 감사·제어 | 제한적 | 상세(CloudTrail 등) |
