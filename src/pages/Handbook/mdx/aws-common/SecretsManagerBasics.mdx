## Parameter Store와의 비교

<KeyTable
  columns={["항목", "Secrets Manager", "SSM Parameter Store"]}
  rows={[
    ["주 용도", "고가치 시크릿(DB PW, API 키)", "구성값 + 가벼운 시크릿"],
    ["타입", "시크릿 문자열 + 버전", "String / StringList / SecureString"],
    ["회전 기능", "자동 회전(템플릿 제공)", "없음(직접 구현)"],
    ["가격", "호출당 과금(유료)", "대부분 무료/저렴"],
  ]}
/>

**실무 패턴**

- **고가치 시크릿 (DB 패스워드, JWT Secret, 3rd party API 키)**  
  → Secrets Manager + KMS + (필요 시) 자동 회전
- 일반 구성값, 환경별 스위치, 덜 민감한 값  
  → Parameter Store(String / SecureString)
