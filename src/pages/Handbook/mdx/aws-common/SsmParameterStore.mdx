## Secrets Manager와의 역할 분리

<KeyTable
  columns={["항목", "Parameter Store", "Secrets Manager"]}
  rows={[
    ["주 용도", "구성값 + 가벼운 시크릿", "고가치 시크릿(DB PW, API 키)"],

/>

**실무 패턴**

- 환경별 구성값(Endpoint, Feature flag 등): Parameter Store(String)
- 덜 민감하지만 암호화가 필요한 값: Parameter Store(SecureString + KMS)
- 고가치 시크릿(회전 필요): Secrets Manager + KMS
