## Action Group

Action Group은 Agent가 **외부 세계와 상호작용**하기 위한 도구 세트이다. 각 Action Group은 Lambda 함수와 연결되어 실제 작업을 수행한다.

## Knowledge Base

Knowledge Base는 **S3에 저장된 문서를 자동으로 RAG 파이프라인으로 구성**해주는 관리형 서비스이다.

<MermaidDiagram
  code={`flowchart LR
  subgraph setup ["설정 (1회)"]
    S3["S3 버킷\n(PDF, HTML, TXT 등)"]
    SYNC["자동 동기화\n(청킹 · 임베딩)"]
    VDB["벡터 DB\n(OpenSearch Serverless\n/ Pinecone / Aurora)"]
    S3 --> SYNC --> VDB
  end

  subgraph query ["질의 (매 요청)"]
    Q["사용자 질문"]
    EMB["질문 임베딩"]
    SEARCH["벡터 검색"]
    CTX["관련 문서 + 질문"]
    FM["FM 생성"]
    ANS["근거 기반 답변\n+ 출처"]
    Q --> EMB --> SEARCH --> CTX --> FM --> ANS
  end

  VDB -.-> SEARCH`}
/>
