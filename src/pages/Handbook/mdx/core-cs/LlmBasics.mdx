## LLM 동작 원리

<MermaidDiagram
  code={`flowchart LR
  INPUT["입력 텍스트"]
  TOK["토크나이저\n(텍스트 → 토큰)"]
  EMB["임베딩\n(토큰 → 벡터)"]
  TF["Transformer 레이어\n(Self-Attention × N)"]

  OUT["출력 토큰"]

  INPUT --> TOK --> EMB --> TF --> PRED --> OUT
