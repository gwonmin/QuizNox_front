## 에코시스템 메모: MCP

Model Context Protocol(MCP)은 LLM 클라이언트가 로컬·원격 도구·데이터 소스에 붙기 위한 개방형 규약으로, 에이전트 구현 시 REST 래핑을 일관되게 쓰기 좋다. 표준 자체가 곧 보안(권한·감사)을 대신해 주지는 않으므로 허용 도구 화이트리스트·OAuth·감사 로그는 애플리케이션에서 설계한다.
