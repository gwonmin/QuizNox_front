## 4. 접근 흐름 도식

- **Key-Value**: 앱 → 키로 조회 → 값 반환.
- **문서 DB**: 앱 → 문서 ID·쿼리 → 문서(필드·중첩) 반환.

<MermaidDiagram
  code={`flowchart LR
  subgraph kv [Key-Value]

  end
  subgraph doc [Document]
    Q[쿼리] --> D[문서]
  end
  App[앱] --> kv
  App --> doc`}
/>
