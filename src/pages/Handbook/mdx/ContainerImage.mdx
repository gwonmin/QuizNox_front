## 이미지 핵심 개념

**한 줄 흐름:** Dockerfile 빌드 → 레이어 단위로 이미지 생성 → 레지스트리 푸시 → 노드에서 풀 시 레이어 캐시로 효율적으로 받아와 컨테이너 실행에 사용한다.

## 이미지 빌드·배포 흐름

<MermaidDiagram
  code={`flowchart LR
  df["Dockerfile"] --> build["빌드"]
  build --> layers["레이어 이미지"]
  layers --> push["레지스트리 푸시"]
  push --> reg["Registry"]
  reg --> pull["노드에서 Pull"]
  pull --> run["컨테이너 실행"]`}
/>
