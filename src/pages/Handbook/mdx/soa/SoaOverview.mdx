## 핵심 서비스 (빈출 주제)

- **CloudWatch**: 커스텀 지표, 로그 그룹/스트림, Logs Agent, 알람 → SNS 알림·EC2 자동 복구
- **AWS Config·CloudTrail**: 리소스 변경·규정 준수, API 호출 감사
- **VPC·네트워킹**: 서브넷·라우팅·IGW, Security Group vs NACL 트러블슈팅
- **SSM**: Patch Manager, Run Command, Session Manager, Parameter Store
- **CloudFormation**: 스택 생성/업데이트/삭제, StackSets
- **고가용성·복구**: ASG 헬스체크·스케일 정책, ELB, RDS Multi-AZ·Read Replica, S3·EBS 백업
