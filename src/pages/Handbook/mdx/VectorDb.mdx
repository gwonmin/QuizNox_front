## 4. 흐름 도식

- **저장**: 원본 → 임베딩 → 벡터 저장소. **조회**: 쿼리 → 임베딩 → 유사도 Top-K → 결과.

<MermaidDiagram
  code={`flowchart LR
  Q[쿼리] --> E[임베딩]
  E --> V[벡터 저장소]
  V --> K[유사도 Top-K]
  K --> Res[결과]`}
/>

## 5. 예: RAG 문서 유사도 검색

- 문서 청크·질의를 임베딩해 저장·조회. 유사 청크 Top-K를 맥락으로 사용.

<MermaidDiagram
  code={`flowchart LR
  subgraph query [쿼리]
    Q[질의 텍스트]
    Q --> E[임베딩]
    E --> VQ[쿼리 벡터]
  end
  subgraph search [유사도 검색]
    VQ --> Dist[거리 계산]
    VS[저장된 벡터들] --> Dist
    Dist --> TopK[Top-K 결과]
  end`}
/>
